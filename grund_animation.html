Animation

I animation forløbet fik vi til opgave at designe og programmere et grundlæggende spil. Vi startede med udvikling af et koncept og valg af en designstil. Ved hjælp af Adobe Illustrator skabte vi spillets assets. Vi implementerede derefter vores design ved hjælp af HTML, CSS og JavaScript, samt vores forståelse af aktivitetsdiagrammer og tilstandsmaskinediagrammer for at bringe spillet til live med animationer.

Grundlæggende web introduceret os til HTML, CSS via VScode samt opsættelse af vores egen domæne. Temaet var opdelt i to opgaver: Studiestartsprøven, som gik ud på at bygge en hjemmeside baseret på en forudsat wireframe, layoutdiagram og indhold. Anden opgave som var vores Emnesite, hvor vi brugte vores nyelærte kompetencer til at skabe en hjemmeside med eget valgt emne.
